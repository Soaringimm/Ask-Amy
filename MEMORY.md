# MEMORY.md — Ask Amy Support 记忆

## 产品概览
Ask Amy 是专业咨询服务平台，核心功能：
- **知识库** — 浏览和搜索专业建议文章
- **个性化咨询** — 提交咨询请求，获得定制报价和专业回复
- **自愿打赏** — 支持微信/支付宝打赏
- **Guest 模式** — 无需注册，提供姓名和邮箱即可使用
- **用户账户** — 注册后可收藏文章、管理个人资料

## 平台角色
- **访客/Guest** — 浏览知识库、提交咨询（只需姓名+邮箱）
- **注册用户** — 额外享有收藏、个人资料等功能
- **管理员** — 管理文章、咨询、报价
